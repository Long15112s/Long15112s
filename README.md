### 🧩 [Buraitoraito — Deduktiver Solver](https://github.com/Long15112s/bachelor-buraitoraito)
> **Bachelorarbeit @ TU Berlin**

Ein deduktiver Solver für das Logikrätsel **Buraitoraito**, inklusive automatischer Instanzgenerierung und statistischer Auswertung.

- ⚙️ **Solver** in C++ (CMake, C++17) mit Debug-Modus und LoE-Stufenanalyse
- 🐍 **Instanzgenerierung** via Python — konfigurierbar nach Schwierigkeitsgrad (leicht → sehr schwer)
- 📊 **Auswertung & Plots** mit Python-Skripten (CSV-basiert)
